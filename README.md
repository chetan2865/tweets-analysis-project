# tweets-analysis-project
This project analyzes tweet sentiment . It preprocesses text, categorizes sentiment (negative, neutral, positive), visualizes distribution with a pie chart and word cloud, and trains/evaluates a Naive Bayes classifier for sentiment prediction, achieving over 90% accuracy.
# 🐦 Twitter Sentiment Analysis using Google Colab

This project analyzes sentiment in Twitter data using Natural Language Processing (NLP) techniques and Python libraries.  
It was developed as part of a  **data science project** to classify tweets as **positive, negative, or neutral** based on textual content.

---

## 📘 Project Overview
The goal of this project is to understand public opinion and emotional tone from tweets.  
Using preprocessing techniques and sentiment classification models, the notebook demonstrates end-to-end workflow — from data cleaning to visualization.

---

## 🧠 Key Features
- Data loading and exploration of Twitter dataset  
- Text preprocessing (tokenization, stopword removal, stemming)  
- Sentiment analysis using libraries such as **TextBlob**, **VADER**, or custom models  
- Visualization of sentiment distribution with **Matplotlib** / **Seaborn**  
- Handles coded emojis and special characters in tweets  
- Works seamlessly in **Google Colab**

## 🚀 How to Run the Notebook
1. Open in Google Colab

2. Run all cells (`Runtime → Run all`)  
3. Upload or link your dataset (CSV with tweet text)  
4. View sentiment results and graphs
